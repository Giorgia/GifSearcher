# GIF Searcher

## Requirements

* Xcode: 7 
* SDK: iOS 9
* CocoaPods 0.39

## Installation

* CocoaPods

	CocoaPods manages dependencies for your Xcode projects.
	
	- Install [CocoaPods](http://cocoapods.org/) `$ gem install cocoapods`
	- Run setup `$ pod setup`
    - Install pods `$ pod install`
